# Dubhe 文档版本

Dubhe 支持多版本文档，方便用户查阅历史与最新内容。

## 可用版本

- [v1 文档](./v1/README.md)
- [最新文档](../README.md)

## 如何切换版本

- 通过文档页面右上角/侧边栏的"版本切换"下拉菜单选择所需版本。
- 访问 `/docs/versions/v1/` 可查看 v1 版本文档。
- 访问 `/docs/` 或 `/docs/README.md` 查看最新主线文档。

## 版本化策略

- 每个大版本（如 v1, v2）单独维护一套文档目录。
- 旧版本文档只做安全/勘误维护，主力开发在最新版本。
- 版本切换组件可集成到前端/文档站点导航。
