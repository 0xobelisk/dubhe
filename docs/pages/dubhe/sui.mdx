## Sui

![sui](/sui/sui.png)

Sui is a smart-contract platform with high throughput, low latency, and an asset-oriented programming model based on Move.

## Dubhe on Sui

- [CLI](/dubhe/sui/cli) — command-line tools for building, testing, and deploying Dubhe packages on Sui.
- [Client](/dubhe/sui/client) — TypeScript/JavaScript SDK for transactions, queries, and optional Channel or indexer integration.
- [Resources](/dubhe/sui/contracts/resources) — declare `resources` in `dubhe.config.ts`, generate Move modules, and model on-chain storage.
- [Tutorials](/dubhe/sui/tutorials) — guided walkthroughs (some examples may predate the current `resources` model; prefer the Resources section for authoritative config).
- [Examples](/dubhe/sui/examples) — reference repositories and sample projects.
