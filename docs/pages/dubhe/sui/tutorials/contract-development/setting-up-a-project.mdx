# Setting up a project

Use the contract template when you only need Move-side development first.

## 1) Create project

```bash
pnpm create dubhe
```

Choose:

- chain: `sui`
- platform/template: `contract`

Then:

```bash
cd <your-project>
pnpm install
```

## 2) Initial structure

A fresh contract template contains:

```text
<project>/
├── dubhe.config.ts
├── package.json
└── tsconfig.json
```

## 3) Generate Move package from config

```bash
pnpm run gen
# equivalent: pnpm dubhe schemagen
```

After generation, you will get:

```text
src/<name>/
├── Move.toml
└── sources
    ├── codegen
    ├── scripts
    ├── systems
    └── tests
```

`<name>` comes from `dubhe.config.ts` (`name` field).

## 4) Open in editor

```bash
code .
```
