# Publish a contract

After writing contract logic, publish the generated Move package to the target Sui network.

## Localnet publish

```bash
pnpm dubhe publish --network localnet
```

Expected result:

- framework package publication succeeds
- your app package publication succeeds
- schema object IDs are printed
- deploy logs are written under `.history/sui_localnet/`

## Testnet publish

```bash
pnpm dubhe check-balance --network testnet
pnpm dubhe faucet --network testnet
pnpm dubhe publish --network testnet
```

## Persist deployment config

Store deployment output for downstream scripts/services:

```bash
pnpm dubhe config-store --network localnet --output-ts-path ./deployment.ts
```

Use `testnet` or `mainnet` when targeting those environments.

## Verify publication output

From the publish logs, capture:

- package ID
- upgrade capability ID
- created schema object IDs

These IDs are required for client integration and production services.
