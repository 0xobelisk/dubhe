# Dubhe Examples

## In-Repo Example (Recommended)

- `examples/sui/debug-workbench-case`
- `examples/sui/session-flow-web`

This example demonstrates a complete local debugging workflow:

- artifact collection (`debug:collect`)
- visual web debugger (`debug:web`)
- VSCode trace launch (`debug:open`)
- terminal debugger (`debug:tui`)
- trace A/B compare + gas heatmap + exception clustering
- conditional breakpoints + watch vars + hit count
- one-click replay/debug-open + report export (`JSON`/`MD`/`HTML`)

Quick start:

```bash
cd examples/sui/debug-workbench-case
pnpm install
pnpm run debug:collect
pnpm run debug:web
```

Open `http://localhost:4173`.

### Session Flow Web (Passkey/Wallet -> Delegate Key)

This example demonstrates session registration + delegate execution flow in one frontend:

- owner creates `SessionCap` with scope/ttl/max-uses
- delegate submits nonce-protected writes
- cap revoke in the same flow

Quick start:

```bash
cd examples/sui/session-flow-web
pnpm install
pnpm dev
```

Open `http://127.0.0.1:4310`.
