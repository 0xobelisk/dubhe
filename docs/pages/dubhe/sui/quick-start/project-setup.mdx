# Project setup

Create a Dubhe project from a template with:

```bash
pnpm create dubhe
```

The interactive prompt has two key steps.

**Step 1. Input your projectName.**

First, enter a project name. Dubhe creates a directory with that name.

For demonstration purposes, we'll define it here as **my_first_project**

**Step 2. Pick your platform.**

Then choose a template.

Current Sui templates include:

- **101** (quick start full stack)
- **web** (frontend-focused template)
- **contract** (contract-first template)
- **eve-extension** (EVE extension scaffold)
- **eve-builder** (EVE builder smart assembly workflow)

Any template works, but examples below use **101**.

**Start my first 101 project**

After selecting a template, move into the project directory:

1. install node modules

```bash
pnpm install
```

2. check environment

```bash
pnpm dubhe doctor
```

3. starting the development server

```bash
pnpm run dev
```

Once the project has been successfully launched, please open `http://localhost:3000` in your browser and enjoy your first obelisk project!

## EVE Builder Template

If you choose `eve-builder`, run:

```bash
pnpm install
pnpm run type-check
pnpm run build:smart-gate
```
