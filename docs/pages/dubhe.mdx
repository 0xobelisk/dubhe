# Introduction

## Dubhe overview

- [Quick start](/dubhe/sui/quick-start) — scaffold a project with `pnpm create dubhe` and follow the end-to-end workflow.
- [Resources](/dubhe/sui/contracts/resources) — declare on-chain state in `dubhe.config.ts` under `resources`, run code generation, and align with indexer-friendly events (ORM-style ergonomics on top of Move).
- [CLI](/dubhe/sui/cli) — build, test, publish, and operate a Dubhe project from the terminal.
- [Client](/dubhe/sui/client) — interact with contracts from TypeScript or JavaScript.
