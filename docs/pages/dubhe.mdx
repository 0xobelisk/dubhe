# Introduction

## Dubhe Architecture?

- [create-dubhe](/dubhe/sui/quick-start) bootstraps Dubhe template projects quickly.
- [Schema](/dubhe/sui/schemas) provides typed on-chain storage abstractions and [ORM](https://en.wikipedia.org/wiki/Object-relational_mapping)-style workflows for Move.
- [Cli](/dubhe/sui/cli) supports build, test, publish, and debugging workflows.
- [Client](/dubhe/sui/client) connects applications to Dubhe projects at runtime.
