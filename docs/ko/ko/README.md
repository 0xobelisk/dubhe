## 🔄 문서 업데이트

이 문서는 각 릴리스에서 적극적으로 유지보수되고 업데이트됩니다. 문서에 기여하려면:

1. 저장소 포크
2. `docs/ko/` 디렉토리에서 변경사항 작성
3. Pull Request 제출
4. [기여 가이드](./contributing/guide.md) 따르기

## 📝 문서 표준

- 모든 문서는 Markdown 형식
- 코드 예제는 다국어 지원 포함
- 유용한 곳에 스크린샷과 다이어그램 포함
- 버전 정보를 명확하게 표시
- 링크를 정기적으로 확인하고 업데이트
